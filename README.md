# 2DMatrix
A 2D matrix program to illustrate how arrays work!

Matrix:
 - - - - -
 - - - - -
 - - - - -
 - - - - -
 - - - - -


Left to right diagonal matrix:
 X - - - -
 - X - - -
 - - X - -
 - - - X -
 - - - - X


Right to left diagonal matrix:
 - - - - X
 - - - X -
 - - X - -
 - X - - -
 X - - - -


Upper diagonal matrix:
 X X X X X
 - X X X X
 - - X X X
 - - - X X
 - - - - X


Lower diagonal matrix:
 - - - - -
 X - - - -
 X X - - -
 X X X - -
 X X X X -


Line after line matrix:
 X X X X X
 - - - - -
 X X X X X
 - - - - -
 X X X X X


Column after column matrix:
 X - X - X
 X - X - X
 X - X - X
 X - X - X
 X - X - X


Chess matrix:
 X - X - X
 - X - X -
 X - X - X
 - X - X -
 X - X - X
